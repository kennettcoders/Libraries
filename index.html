Welcome to the Kennett Coders' GitHub Page

If you click on the "View on GitHub" Button, you will be able to see all the code that we have written this year, and in years past. The goal of our GitHub is to allow other members of our team, and for other people, to use code that has been written to solve problems. We have two branches on our GitHub; the [master] branch and the [gh-pages] branch(which you are viewing right now). If in the future, we have more branches, we will update this page with the information on those branches.

[master] branch

This is where all our files and code is located. It is organized by competition year, and then team number and name.

[gh-pages] branch

This is the website face. any file in here will be rendered for web viewing, like this page.
